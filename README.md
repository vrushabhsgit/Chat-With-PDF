# Chat-With-PDF 

An Applied AI project that enables users to interact with PDF documents through natural language conversations.

Instead of manually searching through lengthy documents, users can upload a PDF and ask questions in plain English. The system retrieves relevant information from the document and generates context-aware responses using Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG).

## Features

- PDF document ingestion
- Semantic search over document content
- Natural language conversations with PDFs
- LLM-powered question answering
- Context-aware responses
- Retrieval-Augmented Generation (RAG)

## Use Cases

- Research papers
- Technical documentation
- Books and eBooks
- Legal documents
- Company reports
- Study materials

## How It Works

1. Upload a PDF document.
2. Extract and preprocess text.
3. Convert text into embeddings.
4. Store embeddings in a vector database.
5. Retrieve relevant context for user queries.
6. Generate accurate answers using an LLM.

## Getting Started

```bash
git clone https://github.com/vrushabhsgit/Chat-With-PDF.git
cd Chat-With-PDF
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python index.py
```
